# Professional Email Generator using Generative AI

## Project Description
This project generates highly professional emails using Generative AI.
The user provides the recipient type, purpose, and tone of the email,
and the system automatically generates a formal email.

## Project Files
- professional_email_generator.py
- README.md

## Requirements
- Python 3.9+
- transformers
- torch

## How to Run
pip install transformers torch
python professional_email_generator.py

## Sample Input
Recipient: Manager
Purpose: Request for leave approval
Tone: Formal

## Output
A professionally written email suitable for official communication.

## Academic Use
Suitable for college projects, assignments, and viva.
