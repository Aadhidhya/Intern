
# Max Profit Problem

## Problem Summary
Given limited development time `n`, determine the optimal mix of:
- Theatre (T)
- Pub (P)
- Commercial Park (C)

to maximize earnings.

## How to Run
```bash
python solution.py
```

## Approach
- Dynamic Programming
- At each time unit, choose the building that gives maximum future earnings

## Output
Displays:
- Total profit
- Number of T, P, C built

## Example
Input: n = 13
Output:
T: 2, P: 0, C: 0
Profit: 16500
